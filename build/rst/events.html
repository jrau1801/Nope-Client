events module¶

events.callback(data)¶

Prints data on acknowledgment :param data: data received from server :return: nothing

events.connect()¶

Connect to server :return: nothing

events.create_tournament(num_best_of_matches)¶

Creates a tournament with an emit to the server :param num_best_of_matches: number of max matches :return: nothing

events.disconnect()¶

Disconnect from server :return: nothing

events.game_state(data, _)¶

Prints the current game-state :param data: game-state-data received from server :param _: placeholder :return: nothing

events.game_status(data, _)¶

Prints message and winner :param data: status-data received from server :param _: placeholder :return: nothing

events.join_tournament(tournament_id)¶

Join a tournament with an id :param tournament_id: tournament you want to join :return: nothing

events.leave_tournament()¶

Leave a tournament :return: nothing

events.list_tournaments(data, _)¶

Prints list of tournaments :param data: tournament-list data received from server :param _: placeholder :return: nothing

events.login(name, password)¶

Handles Client-Login :param name: Name of client-player :param password: password of client-player :return: true if successful, false otherwise

events.make_move(data)¶

Initializes ai-player move :param data: turn-data received from server :return: move to make

events.match_info(data, _)¶

Prints match-info :param data: match-info-data received from server :param _: placeholder :return: nothing

events.player_info(data, _)¶

Prints player-info :param data: player-info-data received from server :param _: placeholder :return: nothing

events.registration(name, password, firstname, lastname)¶

Handles Client-Registration :param name: Username of new user :param password: Password of new user :param firstname: Real firstname of User :param lastname: Real lastname of user :return: true if successful, false otherwise

events.start_tournament()¶

Start a tournament :return: true if successful, false otherwise

events.tournament_info(data, _)¶

Prints tournament-info :param data: tournament-info-data received from server :param _: placeholder :return: nothing
